# React & Tailwind Image Gallery

> In this project I integrated Tailwind CSS with React and built an image gallery app with the Pixabay API

## Technologies

- React 17.0.1
- Tailwind CSS 2.0.3
- Postcss 8.3.1
- Autoprefixer 10.2.4
- Pixabay API

## Features

- Search Pixabay for pictures
- Get picture cards with description
